Set-up

This R markdown document contains the complete code for determing drivers of variation of bulk RNA-sequencing files for Brain Integrative Transcriptome Hub (BITHub). All datasets were preprocessed using the code in 01-bulk-preprocess-data.Rmd, important metadata were retained in 03-bulk-preprocess-data.Rmd and deconvolution analysis was performed in 04-bulk-deconvolution.Rmd.

In order to rerun the scripts - make sure to change the directories to the location of where the input files for preprocessing are stored. All directories are defined in each dataset section.

All other files are stored within the data folder of this R workflow directory.

To investigate the impact of biological and technical factors on gene expression, I performed an initial exploratory analysis on all bulk RNA-seq datasets grouped together via a PCA. The count matrices from all bulk RNA-seq datasets were filtered to remove lowly expressed genes. The expression cut-off was selected at 1 RPKM/TPM in at least 10% of the samples

thresh <- function(x) {
      y <- x > 1
      keep <- rowSums(y) > (ncol(y) / 10)
      return(x[keep,])
    }
gtex <- thresh(e$Raw$GTEx)
span <- thresh(e$Raw$BSpan)
seq <- thresh(e$Raw$BSeq)
pe <- thresh(e$Raw$PE)
hdbr <- thresh(e$Raw$HDBR)
common <- intersect(rownames(gtex), rownames(span))
common <- intersect(common, rownames(seq))
common <- intersect(common, rownames(pe))

merge <- cbind(gtex[common,],

variancePartition is a statistical and visualization framework which fits a linear mixed model for each gene and partitions the total variance into the contribution of each variable in the experimental design, plus the residual variance. The fractions sum to 1 for each gene, the variance fractions are easily interpretable across genes and sources of variation.
